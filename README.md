# Megadriven
A game launcher made in electronjs for genesis (works with blastem emulator)
download blast emulator at: https://www.retrodev.com/blastem/
the application saves the last configuration (path to rom folder, path to image folder, and path to emulator)

the rom files must have the same name of the image files.
